## 🚀 如何启动项目

### 步骤1: 安装依赖

```bash
cd c:\Users\Administrator\Documents\Maillionaire
npm install
npm install dompurify  # HTML清理库
```

### 步骤2: 启动开发服务器

```bash
# Web开发模式
npm run dev

# Electron开发模式
npm run electron:dev
```

### 步骤3: 查看UI设计

在浏览器中打开 `design/ui-mockups/` 下的HTML文件
