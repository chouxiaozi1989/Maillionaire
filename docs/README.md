## 📝 文档维护

### 添加新文档

1. 根据文档类型放入对应目录
2. 更新本索引文件
3. 更新 [CHANGELOG.md](07-版本记录/CHANGELOG.md)
4. 如有必要，更新 [README.md](../README.md)

### 修改文档

1. 更新文档内容
2. 修改文档头部的日期和版本号
3. 如果是重要变更，更新 [CHANGELOG.md](07-版本记录/CHANGELOG.md)

### 废弃文档

1. 移动到 [archive/](../archive/) 目录
2. 在原位置添加重定向说明
3. 更新本索引文件
4. 更新相关链接
