# 邮件模板和签名使用指南

## 概述

Maillionaire 提供了强大的邮件模板和签名管理功能，帮助您提高邮件编写效率和专业性。

---

## 邮件模板管理

### 什么是邮件模板？

邮件模板是预先定义好的邮件内容框架，包括：
- **模板名称**：便于识别和选择
- **邮件主题**：可选的预设主题
- **邮件正文**：支持富文本格式的邮件内容

### 创建邮件模板

1. 进入 **设置** → **邮件模板**
2. 点击 **新建模板** 按钮
3. 填写模板信息：
   - 模板名称（必填）：如"工作汇报模板"
   - 邮件主题（可选）：如"本周工作汇报"
   - 邮件内容（必填）：使用富文本编辑器编写邮件模板
4. 点击 **创建** 保存

**示例模板**：

```
模板名称: 工作汇报模板
邮件主题: 本周工作汇报
邮件内容:
---
尊敬的领导：

现将本周工作汇报如下：

一、已完成工作
1. [填写内容]
2. [填写内容]

二、进行中的工作
1. [填写内容]
2. [填写内容]

三、下周计划
1. [填写内容]
2. [填写内容]

此致
敬礼！
```

### 使用邮件模板

1. 在撰写邮件窗口中，点击底部的 **使用模板** 按钮
2. 从下拉菜单中选择想要使用的模板
3. 模板的主题和内容会自动填充到邮件中
4. 根据实际情况修改模板内容
5. 添加收件人后即可发送

### 编辑/删除模板

1. 进入 **设置** → **邮件模板**
2. 在模板列表中找到目标模板
3. 点击 **编辑** 修改模板内容
4. 点击 **删除** 移除不需要的模板（需确认）

## 邮件签名管理

### 什么是邮件签名？

邮件签名是添加在邮件末尾的固定内容，通常包括：
- 姓名
- 职位/头衔
- 公司名称
- 联系方式
- 公司 Logo（可选）

### 创建邮件签名

1. 进入 **设置** → **签名管理**
2. 点击 **新建签名** 按钮
3. 填写签名信息：
   - 签名名称（必填）：如"正式签名"、"个人签名"
   - 签名内容（必填）：使用富文本编辑器编写签名
4. 点击 **创建** 保存

**示例签名**：

```
签名名称: 商务签名
签名内容:
---
此致
敬礼！

张三
高级产品经理
XX科技有限公司
电话: 138-0000-0000
邮箱: zhangsan@example.com
官网: www.example.com
```

```
签名名称: 英文签名
签名内容:
---
Best Regards,

Zhang San
Senior Product Manager
XX Technology Co., Ltd.
Tel: +86 138-0000-0000
Email: zhangsan@example.com
Website: www.example.com
```

### 设置默认签名

1. 进入 **设置** → **签名管理**
2. 在签名列表中找到目标签名
3. 点击 **设为默认** 按钮
4. 默认签名会在回复邮件时自动插入

### 使用签名

#### 方法一：手动插入

1. 在撰写邮件窗口中，点击底部的 **插入签名** 按钮
2. 从下拉菜单中选择想要使用的签名
3. 签名会自动插入到邮件内容末尾

#### 方法二：自动插入

- 当回复邮件时，系统会自动在邮件开头插入默认签名

### 编辑/删除签名

1. 进入 **设置** → **签名管理**
2. 在签名列表中找到目标签名
3. 点击 **编辑** 修改签名内容
4. 点击 **删除** 移除不需要的签名（需确认）

## 最佳实践

### 模板使用建议

1. **创建常用模板**：
   - 工作汇报模板
   - 请假申请模板
   - 会议邀请模板
   - 客户问候模板
   - 跟进邮件模板

2. **模板命名规范**：
   - 使用清晰易懂的名称
   - 可以按用途分类（如"客户-问候"、"内部-汇报"）

3. **定期更新**：
   - 根据实际使用情况优化模板内容
   - 删除不再使用的过时模板

### 签名使用建议

1. **创建多个签名**：
   - 正式商务签名（对外）
   - 内部沟通签名（对内）
   - 个人邮件签名（非正式）
   - 英文签名（国际业务）

2. **签名内容建议**：
   - 保持简洁专业
   - 包含必要的联系方式
   - 避免过多装饰
   - 注意字体和颜色的统一性

3. **格式规范**：
   - 建议使用分隔线（---）与正文分开
   - 字号不宜过大（建议 12-14px）
   - 颜色以深色为主，避免花哨

## 常见场景示例

### 场景1：发送工作汇报

1. 点击 **撰写邮件**
2. 选择 **使用模板** → **工作汇报模板**
3. 填写收件人和具体内容
4. 点击 **插入签名** → **正式签名**
5. 发送邮件

### 场景2：回复客户邮件

1. 点击邮件的 **回复** 按钮
2. 系统自动插入默认签名
3. 编写回复内容
4. 如需使用模板，可选择 **使用模板** → **客户回复模板**
5. 发送邮件

### 场景3：发送会议邀请

1. 点击 **撰写邮件**
2. 选择 **使用模板** → **会议邀请模板**
3. 修改会议时间、地点等信息
4. 添加多个收件人
5. 点击 **插入签名** → **商务签名**
6. 发送邮件

## 数据存储说明

- **模板数据**：存储在 `templates.json` 文件中
- **签名数据**：存储在 `signatures.json` 文件中
- **存储位置**：
  - Electron 环境：`userData` 目录
  - 浏览器环境：`localStorage`

## 注意事项

1. **备份重要模板和签名**
   - 定期导出重要模板和签名内容
   - 避免因系统问题导致数据丢失

2. **富文本格式**
   - 模板和签名支持富文本格式（HTML）
   - 支持文字样式、颜色、链接、图片等

3. **隐私保护**
   - 模板和签名存储在本地
   - 不会上传到服务器
   - 确保个人信息安全

4. **性能考虑**
   - 建议模板数量控制在 20 个以内
   - 建议签名数量控制在 5 个以内
   - 避免在签名中使用过大的图片
