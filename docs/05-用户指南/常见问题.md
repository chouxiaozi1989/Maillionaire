# Maillionaire 常见问题 (FAQ)

> 更新日期：2025-11-07
> 版本：v1.1.0

---

## 📧 账户相关

### Q1: 支持哪些邮箱？

**A:** Maillionaire 支持以下邮箱：

**OAuth2认证（推荐）：**
- Gmail
- Outlook/Hotmail

**IMAP/SMTP认证：**
- QQ邮箱
- 163邮箱
- 126邮箱
- 新浪邮箱
- 其他支持IMAP/SMTP的邮箱

---

### Q2: 如何添加QQ邮箱？

**A:** QQ邮箱需要使用授权码而非登录密码：

1. 登录QQ邮箱网页版
2. 进入"设置" → "账户"
3. 找到"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"
4. 开启"IMAP/SMTP服务"
5. 生成授权码（记住这个授权码）
6. 在Maillionaire中添加账户时使用授权码作为密码

**服务器配置：**
- IMAP服务器：imap.qq.com
- IMAP端口：993
- SMTP服务器：smtp.qq.com
- SMTP端口：465

详见：[QQ邮箱添加指南](./QQ邮箱添加指南.md)

---

### Q3: Gmail账户无法登录？

**A:** 可能的原因和解决方法：

1. **未启用Gmail API**
   - 需要在Google Cloud Console中启用Gmail API
   - 配置OAuth2客户端ID
   - 详见：[OAuth2配置指南](../02-开发文档/OAuth2配置指南.md)

2. **网络连接问题**
   - Gmail需要访问Google服务
   - 如果在中国大陆，请配置代理
   - 详见：[代理配置使用指南](../代理配置使用指南.md)

3. **OAuth2回调失败**
   - 检查回调地址配置是否正确
   - 默认回调地址：`http://localhost:5173/oauth/callback`

---

### Q4: 如何删除账户？

**A:** 删除账户步骤：

1. 点击主界面左上角的账户选择器
2. 点击要删除的账户右侧的"删除"按钮
3. 确认删除操作

**注意：** 删除账户会同时清除：
- 该账户的所有邮件数据
- 该账户的文件夹数据
- 该账户的配置信息

## 📬 邮件收发

### Q5: 为什么邮件没有按文件夹分类？

**A:** 已在v1.1.0版本修复此问题。确保：

1. 已同步文件夹（点击主界面的同步按钮）
2. 邮件拉取时会自动按文件夹分类
3. QQ、163、126等邮箱的中文文件夹名称会自动映射

如果问题仍存在：
1. 重启应用
2. 重新同步文件夹
3. 重新拉取邮件

---

### Q6: 如何配置每次拉取的邮件数量？

**A:** 配置拉取邮件数量：

1. 进入"设置" → "通用设置"
2. 找到"每次拉取邮件数"选项
3. 调整数值（10-200封，默认50封）
4. 点击"保存设置"
5. 下次同步时将使用新配置的数量

**建议：**
- 首次使用：50-100封
- 日常使用：10-50封
- 大批量导入：100-200封

---

### Q7: 邮件操作（已读、星标、删除）没有同步到服务器？

**A:** 检查以下几点：

1. **网络连接**
   - 确保有网络连接
   - 如需代理，检查代理配置

2. **账户状态**
   - 确认账户连接正常
   - OAuth2账户令牌可能过期，尝试重新登录

3. **操作同步**
   - 标记已读/未读：自动同步
   - 加星标/取消星标：自动同步
   - 删除邮件：自动同步到服务器

---

### Q8: 如何发送带附件的邮件？

**A:** 发送带附件的邮件：

1. 点击"写邮件"按钮
2. 填写收件人、主题、正文
3. 点击"添加附件"按钮
4. 选择要上传的文件
5. 等待文件上传完成
6. 点击"发送"

**注意事项：**
- 附件大小限制：一般为25MB（取决于邮箱服务商）
- 支持多个附件
- 上传失败请检查网络连接

## 🌐 代理配置

### Q11: 如何配置代理？

**A:** 代理配置步骤：

1. 进入"设置" → "代理设置"
2. 启用代理开关
3. 选择代理协议（推荐SOCKS5）
4. 输入代理服务器地址和端口
   - Clash：127.0.0.1:7890
   - V2Ray：127.0.0.1:1080
5. 如需认证，启用认证并输入用户名密码
6. 点击"测试连接"验证
7. 保存设置并重启应用

详见：[代理配置使用指南](../代理配置使用指南.md)

---

### Q12: 代理连接测试失败？

**A:** 常见原因：

1. **代理软件未启动**
   - 确保Clash/V2Ray等代理软件正在运行

2. **端口配置错误**
   - 检查代理软件的监听端口
   - 确认Maillionaire中配置的端口一致

3. **代理协议不匹配**
   - SOCKS5是最常用的协议
   - 某些代理软件可能只支持特定协议

4. **防火墙阻止**
   - 检查防火墙设置
   - 允许Maillionaire访问网络

---

### Q13: 使用代理后Gmail仍无法连接？

**A:** 检查以下几点：

1. **代理是否支持HTTPS**
   - Gmail需要HTTPS连接
   - 确保代理支持HTTPS流量

2. **代理规则**
   - 确认代理规则允许访问Google服务
   - 检查代理软件的规则配置

3. **DNS解析**
   - 某些代理需要配置DNS
   - 尝试使用远程DNS

---

## ⚙️ 设置和配置

### Q14: 修改设置后不生效？

**A:** 可能的原因：

1. **未保存设置**
   - 确保点击了"保存设置"按钮

2. **需要重启应用**
   - 某些设置（如代理）需要重启应用生效

3. **设置未持久化**
   - 检查应用数据目录权限
   - Windows: `C:\Users\{用户名}\AppData\Roaming\Maillionaire\`

---

### Q15: 如何备份数据？

**A:** 数据备份方法：

1. **定位数据目录**
   - Windows: `C:\Users\{用户名}\AppData\Roaming\Maillionaire\`
   - macOS: `~/Library/Application Support/Maillionaire/`
   - Linux: `~/.config/Maillionaire/`

2. **备份文件**
   - 复制整个Maillionaire目录
   - 包含账户、邮件、通讯录等所有数据

3. **恢复数据**
   - 将备份的文件复制回数据目录
   - 重启应用

## 🔧 故障排除

### Q16: 应用无法启动？

**A:** 故障排除步骤：

1. **检查Node.js版本**
   ```bash
   node --version
   # 需要 >= 16.x
   ```

2. **重新安装依赖**
   ```bash
   npm install
   ```

3. **清除缓存**
   ```bash
   npm cache clean --force
   ```

4. **查看错误日志**
   - 开发模式：查看终端输出
   - 生产模式：查看日志文件

---

### Q17: Electron应用黑屏或白屏？

**A:** 可能的解决方法：

1. **清除应用数据**
   - 删除数据目录（备份后）
   - 重启应用

2. **禁用硬件加速**
   - 在electron/main.js中添加：
   ```javascript
   app.disableHardwareAcceleration()
   ```

3. **更新显卡驱动**
   - 确保显卡驱动为最新版本

---

### Q18: IMAP连接超时？

**A:** 解决方法：

1. **检查网络连接**
   - 确保可以访问邮箱服务器
   - 测试网络稳定性

2. **使用代理**
   - 如果在限制网络环境，配置代理

3. **增加超时时间**
   - 修改IMAP连接超时配置
   - 默认为30秒

4. **减少批量操作数量**
   - 减少每批次拉取的邮件数量
   - 避免一次性拉取太多邮件

## 📱 功能使用

### Q19: 如何使用邮件模板？

**A:** 邮件模板使用方法：

1. **创建模板**
   - 进入"设置" → "邮件模板"
   - 点击"新建模板"
   - 填写模板名称、主题和正文
   - 保存模板

2. **使用模板**
   - 撰写邮件时点击"使用模板"
   - 选择需要的模板
   - 模板内容自动填充
   - 根据实际情况修改后发送

详见：[邮件模板和签名使用指南](./邮件模板和签名使用指南.md)

---

### Q20: 如何设置默认签名？

**A:** 设置默认签名：

1. 进入"设置" → "签名管理"
2. 创建或编辑签名
3. 勾选"设为默认签名"
4. 保存签名

默认签名会在以下情况自动插入：
- 撰写新邮件时
- 回复邮件时
- 转发邮件时

## 💡 使用技巧

### Q21: 如何快速切换账户？

**A:** 快速切换账户：

1. 点击主界面左上角的账户选择器
2. 从下拉列表中选择要切换的账户
3. 系统会自动加载该账户的邮件和文件夹

---

### Q22: 如何批量处理邮件？

**A:** 批量处理邮件：

1. 点击工具栏的"多选"按钮进入多选模式
2. 勾选要处理的邮件
3. 使用批量操作按钮：
   - 全选：选择当前页面所有邮件
   - 标为已读：批量标记为已读
   - 删除：批量移至回收站
4. 完成后点击"取消多选"退出多选模式

---

### Q23: 如何管理通讯录？

**A:** 通讯录管理：

1. **添加联系人**
   - 进入通讯录页面
   - 点击"添加联系人"
   - 填写联系人信息
   - 保存

2. **分组管理**
   - 创建分组（同事、客户、朋友等）
   - 将联系人添加到分组

3. **快速发邮件**
   - 点击联系人卡片上的邮件图标
   - 自动打开撰写窗口并填充收件人

4. **导入导出**
   - 支持CSV格式
   - 批量导入/导出联系人
